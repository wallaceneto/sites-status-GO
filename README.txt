- MONITORADOR DE SITES -

Programa em console para realizar o monitoramento de sites selecionados e reportar 
o status atual de acesso. Programa simples para aprender a sintaxe da linguagem GO. 

O link dos sites que serão monitorados devem estar presentes no arquivo "sites.txt".
O programa gera um arquivo "logs.txt" que armazena o resultado do monitoramento, 
guardando os dados: data e hora da consulta; nome do site; se o site se encontra online
ou o erro que ele apresenta. O programa está setado para a cada monitoramento realizar 
3 consultas com intervalos de 5 segundos entre elas.

O código desse programa seguiu o curso "Go: Iniciando com a linguagem do Google", 
disponivel via assinatura no site da Alura. Os sites presentes no arquivo
"sites.txt" são os mesmos dos apresentados nas aulas. Essa versão corresponde a
versão equivalente obtida após a conclusão do curso, e não à versão final. 
Modificações ainda serão feitas para refinar o programa da maneira como eu achar
melhor.
